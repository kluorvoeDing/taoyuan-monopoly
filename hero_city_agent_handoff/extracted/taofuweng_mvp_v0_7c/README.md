## 啟動方式

直接用瀏覽器開啟：

```text
index.html
```

不需要安裝 Node.js，不需要 npm install。

## 為什麼資料不是 .json？

因為此版本仍要支援「直接雙擊 index.html 遊玩」。

若使用 `fetch("./data/tiles.json")`，部分瀏覽器在本機 file:// 模式下會被 CORS 限制擋住。

所以 v0.3 先採用：

```text
data/tiles.js
data/characters.js
data/cards.js
data/events.js
data/gods.js
```

每個檔案將資料掛到 `window`，可直接被 `game.js` 使用。

## v0.7b Balance Audit + Effect Rules Patch

```text
本版全面檢討角色個性與支援裝備卡，重點是修正高資金局比例失衡、停留規則不符合直覺、攻防效果過強，以及角色描述與實際效果不一致。
```

### 角色個性平衡

```text
- 綠谷出久：移除起始資金 +10%。改為純粹的指定移動點數能力，固定支付 800。
- 飯田天哉：指定 6 點移動成本提高為 600。
- 八百萬百：創造裝備成本提高為 600。
- 麗日御茶子：免付支援費改為支援費 -50%，避免完全免疫高價據點。
- 爆豪勝己：被動擴建折扣由 -15% 降為 -10%；連升 2 級成本懲罰提高為 +45%，冷卻提高。
- 轟焦凍：S 級支援費加成由 +10% 降為 +8%；冰封效果明確會讓目標下回合再次觸發所在格。
- 奮進人：No.1 事務所支援費加成由 +30% 降為 +20%；主動加成由 ×2 降為 ×1.5。
- 切島銳兒郎：C 級進駐折扣由 -10% 降為 -8%。
```

### 卡片效果平衡

```text
- 定點部署：明確會再次觸發目前所在格；若在他人據點上，需要再次繳支援費。
- 捕縛布 / 冰封：被停留者下回合開始時會再次觸發所在格，再跳過移動。
- 無重力漂浮：免付改為支援費 -50%。
- 英雄廣告看板：支援費加成由 ×2 降為 ×1.5。
- 校方稽核：10% 改為 5%，上限 5000。
- 媒體風波：干擾由 3 回合 -50% 改為 2 回合 -40%。
- 據點防線：防守由 4 回合改為 3 回合。
- 支援科裝備：擴建折扣由 -30% 改為 -25%。
- 臨時執照：每回合只能實際使用一次。
```
