# v0.7c 角色與卡片平衡盤點

## 核心修正

```text
- 綠谷出久不再有任何起始資金加成。
- 停留 / 停回合效果會在下回合開始時再次觸發所在格。
- 如果玩家被停在他人據點上，下回合開始會再次支付支援費。
- 完全免付型效果改成折扣型，避免高價據點被完全無效化。
- 遊戲內不再顯示「移除某效果」「原本如何」這類與遊戲無關的修改紀錄。
```
